# CLI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the `vxllm` CLI with 7 commands: serve, pull, run, list, ps, rm, info. Uses citty for command definitions, direct inference imports for offline commands, and the Hono server for serve.

**Architecture:** `apps/cli` package with citty defineCommand. Commands import directly from `@vxllm/inference` (no server needed for pull/list/rm/info/run). `serve` starts the existing Hono server. `run` streams tokens to stdout with terminal markdown.

**Tech Stack:** citty (UnJS), consola, @vxllm/inference, @vxllm/llama-provider, AI SDK streamText, Bun

**Spec:** `docs/superpowers/specs/2026-03-21-cli-design.md`

```typescript
import { detectHardware } from "@vxllm/inference";
import consola from "consola";

export default defineCommand({
  meta: { name: "info", description: "Show hardware profile and recommendations" },
  args: {
    json: { type: "boolean", description: "JSON output", default: false },
  },
  async run({ args }) {
    const hw = await detectHardware();
    if (args.json) {
      console.log(JSON.stringify(hw, null, 2));
      return;
    }
    consola.info(`Platform:    ${hw.platform} (${hw.arch})`);
    consola.info(`GPU:         ${hw.gpu.name} (${hw.gpu.vendor}) — ${formatBytes(hw.gpu.vramBytes)} VRAM`);
    consola.info(`CPU:         ${hw.cpu.model} — ${hw.cpu.physicalCores} cores`);
    consola.info(`RAM:         ${formatBytes(hw.ram.totalBytes)} total, ${formatBytes(hw.ram.availableBytes)} available`);
  },
});
```

## Task 3: Implement pull command (with progress bar)

**Files:**
- Create: `apps/cli/src/commands/pull.ts`

- [ ] **Step 1: Implement pull command**

```typescript
export default defineCommand({
  meta: { name: "pull", description: "Download a model" },
  args: {
    name: { type: "positional", description: "Model name (e.g., qwen2.5:7b)", required: true },
    variant: { type: "string", description: "Quantization variant" },
    force: { type: "boolean", description: "Re-download even if exists", default: false },
  },
  async run({ args }) {
    const registry = new Registry();
    await registry.load();

    const model = await registry.resolve(args.name);
    if (!model) { consola.error(`Model not found: ${args.name}`); process.exit(1); }

    consola.start(`Pulling ${model.name} (${model.variant}, ${formatBytes(model.sizeBytes)})...`);

    const dm = new DownloadManager(registry);
    const progress = await dm.pull(args.name, { variant: args.variant });

    // Poll progress and render progress bar
    const interval = setInterval(() => {
      const p = dm.getProgress(progress.modelId);
      if (!p) return;
      const pct = Math.round(p.progressPct);
      const bar = "━".repeat(Math.floor(pct / 3)) + " ".repeat(30 - Math.floor(pct / 3));
      process.stdout.write(`\r${bar} ${pct}% ${formatBytes(p.downloadedBytes)} / ${formatBytes(p.totalBytes)}  ${formatBytes(p.speedBps)}/s  ETA ${formatDuration(/* eta */)}  `);

      if (p.status === "completed") {
        clearInterval(interval);
        console.log();
        consola.success(`Downloaded ${model.name}`);
      }
      if (p.status === "failed") {
        clearInterval(interval);
        console.log();
        consola.error(`Failed: ${p.error}`);
      }
    }, 500);
  },
});
```

## Task 4: Implement run command (interactive streaming chat)

**Files:**
- Create: `apps/cli/src/commands/run.ts`

- [ ] **Step 1: Implement run command**

The most complex CLI command:

```typescript
export default defineCommand({
  meta: { name: "run", description: "Interactive chat with a model" },
  args: {
    name: { type: "positional", description: "Model name", required: true },
    system: { type: "string", description: "System prompt" },
    temperature: { type: "string", description: "Temperature (0.0-2.0)", default: "0.7" },
  },
  async run({ args }) {
    // 1. Load model
    const registry = new Registry();
    await registry.load();
    const modelInfo = await registry.resolve(args.name);
    // Check if downloaded, if not suggest `vxllm pull`

    const mm = new ModelManager();
    consola.start(`Loading ${args.name}...`);
    const loaded = await mm.load(modelInfo);
    consola.success(`Loaded ${args.name} (${loaded.contextSize} ctx, ${loaded.gpuLayersLoaded} GPU layers)`);

    const provider = createLlamaProvider(mm);
    const model = provider.chat(loaded.sessionId);

    // 2. Readline loop
    const rl = readline.createInterface({ input: process.stdin, output: process.stdout });
    const messages: Array<{ role: string; content: string }> = [];

    if (args.system) {
      messages.push({ role: "system", content: args.system });
    }

    console.log(`\nChat with ${args.name}. Type /bye to exit.\n`);

    const prompt = () => {
      rl.question(">>> ", async (input) => {
        if (input.trim() === "/bye") {
          await mm.disposeAll();
          rl.close();
          return;
        }

        messages.push({ role: "user", content: input });

        const startTime = Date.now();
        const result = streamText({
          model,
          messages: messages.map(m => ({ role: m.role as any, content: m.content })),
          temperature: parseFloat(args.temperature),
        });

        let fullResponse = "";
        for await (const chunk of result.textStream) {
          process.stdout.write(chunk);
          fullResponse += chunk;
        }

        console.log(); // newline after streaming

        // Render with markdown formatting
        // console.log(renderMarkdown(fullResponse));  // optional: re-render with ANSI

        const elapsed = Date.now() - startTime;
        const usage = await result.usage;
        const tokPerSec = usage.completionTokens / (elapsed / 1000);
        console.log(`\x1b[2m${usage.completionTokens} tokens · ${(elapsed/1000).toFixed(1)}s · ${tokPerSec.toFixed(1)} tok/s\x1b[0m\n`);

        messages.push({ role: "assistant", content: fullResponse });
        prompt(); // next turn
      });
    };

    prompt();
  },
});
```

**Key points:**
- `streamText` from `ai` package for streaming
- `process.stdout.write(chunk)` for real-time token display
- Token stats shown in dim text after each response
- `/bye` to exit
- Model loaded once, reused across turns
- Messages accumulated for context

```typescript
export default defineCommand({
  meta: { name: "serve", description: "Start VxLLM server" },
  args: {
    port: { type: "string", description: "Server port", default: "11500" },
    host: { type: "string", description: "Bind host", default: "127.0.0.1" },
    model: { type: "string", description: "Auto-load model on startup" },
  },
  async run({ args }) {
    // Set env vars before importing server
    process.env.PORT = args.port;
    process.env.HOST = args.host;
    if (args.model) process.env.DEFAULT_MODEL = args.model;

    consola.info(`Starting VxLLM server on ${args.host}:${args.port}...`);

    // Import and start the server
    // The server's index.ts default export is { port, hostname, fetch }
    // Bun.serve picks this up automatically when the file is the entry point
    // For CLI: dynamically import and call Bun.serve
    const server = await import("../../server/src/index");

    // The server module already starts via its default export for Bun
    // If it doesn't auto-start, manually call Bun.serve:
    // Bun.serve({ port: parseInt(args.port), hostname: args.host, fetch: server.default.fetch });

    consola.success(`Server running at http://${args.host}:${args.port}`);
    consola.info(`API: http://${args.host}:${args.port}/v1`);
    consola.info(`Health: http://${args.host}:${args.port}/health`);
    consola.info(`Press Ctrl+C to stop`);
  },
});
```

**Note:** The exact mechanism to start the Hono server from the CLI depends on how `apps/server/src/index.ts` exports its server. Read the file during implementation to determine the right approach. It may be as simple as a dynamic import (Bun auto-starts servers from default exports).
